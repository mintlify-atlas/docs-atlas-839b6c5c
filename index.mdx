## Welcome to the TDD Workshop

A hands-on workshop for learning Test-Driven Development by building a React application iteratively. You'll start with a simple "Hello World" app and add functionality iteration by iteration, always following the **Red → Green → Refactor** cycle.

## Workshop Structure

<Steps>
  <Step title="Setup">
    Get your development environment ready with GitHub Codespaces or local installation
  </Step>
  <Step title="Learn TDD">
    Understand the Red-Green-Refactor methodology and test structure
  </Step>
  <Step title="Build Iteratively">
    Complete 5 iterations, each adding new functionality through TDD
  </Step>
  <Step title="Reference">
    Access type definitions, API endpoints, and commands as needed
  </Step>
</Steps>
