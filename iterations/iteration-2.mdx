### Make ProductCard dynamic

Update `src/components/product-card/ProductCard.tsx` to accept props:

```typescript
import './ProductCard.css'
import type { Product } from '../../types'

interface ProductCardProps {
  product: Product
}

export const ProductCard = ({ product }: ProductCardProps) => {
  const formattedPrice = new Intl.NumberFormat('es-ES', {
    style: 'currency',
    currency: 'EUR',
  }).format(product.price)

  return (
    <article className="product-card product-card--card">
      <div className="product-card__image-wrapper">
        <img
          className="product-card__image"
          src={product.image}
          alt={product.displayName}
        />
      </div>
      <div className="product-card__content">
        <h3 className="product-card__name">{product.displayName}</h3>
        <div className="product-card__footer">
          <span className="product-card__price">{formattedPrice}</span>
        </div>
      </div>
    </article>
  )
}
```

### Fetch and render products in Home

Update `src/pages/home/Home.tsx`:

```typescript
import { useEffect, useState } from 'react'
import { ProductCard } from '../../components/product-card/ProductCard'
import type { Product } from '../../types'
import './Home.css'

const Home = () => {
  const [products, setProducts] = useState<Product[]>([])

  useEffect(() => {
    fetch('/products')
      .then(res => res.json())
      .then(data => setProducts(data))
  }, [])

  return (
    <section className="home__products-grid">
      {products.map(product => (
        <ProductCard key={product.id} product={product} />
      ))}
    </section>
  )
}

export { Home }
```

## Price Formatting with Intl

<Note>
The `Intl.NumberFormat` API provides locale-aware number formatting. For Spanish euros:

```typescript
new Intl.NumberFormat('es-ES', {
  style: 'currency',
  currency: 'EUR',
}).format(1.2) // "1,20 €"
```

Notice:
- Decimal separator is a comma (`,`)
- Euro symbol appears after the amount
- Always shows 2 decimal places
</Note>

## Implementation Hints

<Accordion title="How do I create an MSW handler?">

Import the fixtures and create a handler that returns them:

```typescript
import { http, HttpResponse } from 'msw'
import productsFixtures from './products-fixtures.json'

export const handlers = [
  http.get('/products', () => {
    return HttpResponse.json(productsFixtures)
  }),
]
```

</Accordion>

<Accordion title="How do I make a component accept props?">

Define a props interface and destructure the props:

```typescript
interface ProductCardProps {
  product: Product
}

export const ProductCard = ({ product }: ProductCardProps) => {
  // Use product.displayName, product.price, etc.
}
```

</Accordion>

<Accordion title="Why extract price formatting to a variable?">

It keeps JSX clean and makes the formatting logic reusable:

```typescript
const formattedPrice = new Intl.NumberFormat('es-ES', {
  style: 'currency',
  currency: 'EUR',
}).format(product.price)

// Then in JSX:
<span className="product-card__price">{formattedPrice}</span>
```

</Accordion>

## Common Issues

<Tip>
**Issue:** Prices show as "1.2 €" instead of "1,20 €"

**Solution:** Make sure you're using `'es-ES'` locale, not `'en-US'`. The locale determines decimal separators and currency symbol placement.
</Tip>

<Tip>
**Issue:** "productsFixtures is not defined"

**Solution:** Remember to import the fixture at the top of `handlers.ts`:
```typescript
import productsFixtures from './products-fixtures.json'
```
</Tip>
